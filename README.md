# Jialiang Zou — Academic website

Static academic website for `jialiangzou.github.io`.

## Publish with GitHub Pages

1. Upload all files in this directory to the root of the GitHub repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

The site contains Home, Publications, Teaching, and a custom 404 page. It has no build dependencies.

## Update

- Edit `index.html` for biography, research fields, education, and contact information.
- Edit `publications.html` for papers and preprints.
- Edit `teaching.html` for courses and teaching materials.
- Edit `styles.css` for appearance.
